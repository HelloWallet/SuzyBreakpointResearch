Using Components

Sometimes it's not easy to change markup based classes based upon the container that contains the component.

You can bind classes to those components based on the container, but that involves javascript to handle styling. You can override classes as well, but that changes the meaning of clasess like `col-xs-4` to something else!

Better to just use css and handle grids from css.
